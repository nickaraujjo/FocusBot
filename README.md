# 📚 FocusBot

O **FocusBot** é um assistente virtual de estudos desenvolvido em Python com foco em organização, produtividade e aprendizado interativo.

O projeto foi criado com o objetivo de praticar lógica de programação, manipulação de arquivos JSON e desenvolvimento de sistemas interativos no terminal.

---

# 🚀 Funcionalidades

✅ Criar planos de estudo personalizados
✅ Adicionar e visualizar matérias
✅ Sistema de gerenciamento de tarefas
✅ Quiz interativo sobre tecnologia e cibersegurança
✅ Assistente virtual para explicar conceitos
✅ Organização de conteúdos por matéria
✅ Base de conhecimento usando JSON
✅ Interface interativa no terminal

# 🤖 Como funciona

O usuário pode:

* cadastrar matérias
* criar tarefas
* acessar conteúdos organizados
* conversar com o bot
* aprender conceitos de tecnologia
* receber dicas de produtividade

O FocusBot utiliza arquivos JSON para armazenar dados e simular uma base de conhecimento simples.

# ▶️ Como executar

Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/FocusBot.git
```

Acesse a pasta:

```bash
cd FocusBot
```

Execute o projeto:

```bash
python app.py
```
